# 👨‍💻 Portfólio - Levi Rodrigues

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi estruturado para apresentar minhas habilidades, meus trabalhos e um pouco sobre a minha trajetória como **Desenvolvedor Full Stack Júnior**.

## 🚀 Tecnologias Utilizadas

O portfólio foi construído utilizando as seguintes tecnologias base do desenvolvimento web, com código limpo e comentado:

- **HTML5**: Estruturação de conteúdo e semântica da página.
- **CSS3**: Estilização moderna, responsividade e ajustes visuais precisos.
- **JavaScript (Vanilla)**: Interatividade, como o funcionamento do menu mobile, efeitos de scroll no cabeçalho e simulação de envio de formulário de contato.

## 📂 Estrutura de Arquivos

A organização do projeto é simples e direta para facilitar a manutenção:

- `index.html`: Arquivo principal que contém toda a estrutura do portfólio (Cabeçalho, Sobre mim, Habilidades, Projetos, etc).
- `styles.css`: Arquivo com todas as regras de estilo, design e responsividade (adaptação para celular e desktop).
- `script.js`: Arquivo com as lógicas e comportamentos da página.
- `img/`: Pasta para o armazenamento das imagens (fotos de perfil, prints dos projetos, etc).
- `assets/`: Pasta para arquivos e recursos auxiliares (ex: `curriculo.pdf`).

O layout foi totalmente preparado para se adaptar aos mais diversos tamanhos de tela. Desde o menu "hambúrguer" para dispositivos móveis até o redimensionamento de seções e imagens, garantindo que a experiência do usuário seja sempre a melhor possível.

## 💻 Como visualizar este projeto na sua máquina

Sendo um projeto puramente estático (Front-end), a execução é muito simples e não depende de instalação de pacotes:

1. Baixe os arquivos do projeto ou faça o clone do repositório:
   ```bash
   git clone https://github.com/Jander-Levi/Nome-do-Repositorio.git
   ```
2. Abra a pasta baixada.
3. Dê um clique duplo sobre o arquivo `index.html`. Ele será aberto automaticamente no seu navegador web padrão.
*(Dica: Se você utiliza o VS Code, pode usar a extensão "Live Server" para explorar o código com atualizações em tempo real).*
