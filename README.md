# Portfólio Finalizado com Sucesso! 🚀

Acabei de concluir a criação do seu novo Portfólio! O design foi construído usando tecnologias puras (HTML, CSS e JS Vanilla) seguindo a melhor arquitetura de aprendizado para um Dev Júnior. 

Todo o código foi construído com a premissa de **"código limpo"**, com incontáveis comentários explicativos em português ensinando _como_ cada tag e comando funciona.

## O que foi construído e entregue?
- **`index.html`**: Toda a estrutura da página, separada elegantemente em seções (`header`, `hero`, `sobre me`, `habilidades`, `projetos`, `contato` e `footer`).
- **`styles.css`**: Design super premium (Dark Mode) usando Flexbox, Grid Layouts e variáveis de fácil modificação para gerenciar cores.
- **`script.js`**: Os "músculos" do site. Uma programação enxuta e documentada ensinando você a capturar eventos de botão, scroll e botões do celular.

> [!TIP]
> **Como rodar seu projeto na sua máquina:**
> O site está disponível na sua \`Área de Trabalho\`. Dê dois cliques no arquivo \`index.html\` dentro da pasta **Potifolio**. 
> Para visualizar os comentários de aprendizado, abra a pasta do projeto num editor de código como o **VS Code**.

## Próximos Passos (Para você!)
Agora o projeto é seu. Recomendamos as seguintes ações para praticar e deixar o seu portfólio "com a sua cara":
1. **Mudar seu Nome:** Abra o `index.html` e pesquise por `SeuNome`.
2. **Foto de Perfil:** Encontre as tags `<img>` e troque a URL do `src` pelas suas fotos no LinkedIn ou GitHub!
3. **Links Reais:** Troque os `#` nos links dos projetos pelas URLs reais dos seus repositórios e projetos no ar.
